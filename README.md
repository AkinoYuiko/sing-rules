# sing-rules

[![同步状态](https://github.com/AkinoYuiko/sing-rules/actions/workflows/sync-rules.yml/badge.svg)](https://github.com/AkinoYuiko/sing-rules/actions/workflows/sync-rules.yml)
[![License](https://img.shields.io/github/license/AkinoYuiko/sing-rules)](LICENSE)

自动转换的 sing-box Rule Set，每日同步并发布 `.srs` 和 `.json` 文件。

## 获取规则

所有规则均发布于 [`release` 分支](https://github.com/AkinoYuiko/sing-rules/tree/release)，可在该分支查看当前可用的规则文件。

Raw URL：

```text
https://raw.githubusercontent.com/AkinoYuiko/sing-rules/release/<规则名>.srs
https://raw.githubusercontent.com/AkinoYuiko/sing-rules/release/<规则名>.json
```

请将 `<规则名>` 替换为 `release` 分支中的实际文件名（不含扩展名）。
